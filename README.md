# Student Marks Calculator

This is a simple Python program that calculates the **total marks, average marks, and grade** of a student based on the marks entered for different subjects.

# Features
- Takes input for multiple subjects
- Calculates total marks
- Calculates average marks
- Assigns grade based on average
- Displays a student report

# Grade System

| Average Marks | Grade |
|---------------|------|
| 90 and above  | A+ |
| 80 – 89       | A |
| 70 – 79       | B |
| 60 – 69       | C |
| Below 60      | D |
